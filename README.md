# disaster.ai  

# 🚨 AI Disaster Response System

An intelligent disaster management platform that uses AI to analyze emergencies in real-time and suggest immediate actions, severity levels, and required resources.

---

## 🌍 Features

- 🧠 AI-based disaster analysis (text + image)
- 📍 Live location detection using map
- 🎤 Voice input support
- 📊 Dashboard with analytics
- 🗂 History tracking with map markers
- ☁ Hybrid AI:
  - Local AI (Ollama - LLaVA)
  - Cloud AI (Google Gemini)

---

## 🚀 Problem Statement

In emergency situations, people lack immediate guidance and resource coordination. This system helps analyze disasters quickly and provide actionable insights.

---

## 💡 Solution

This platform allows users to:
1. Describe a disaster (text/voice/image)
2. Detect location automatically
3. Use AI to:
   - Identify disaster type
   - Predict severity
   - Suggest actions
   - Recommend resources

---

## 🛠 Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, JavaScript
- AI Models:
  - Ollama (LLaVA - Local AI)
  - Google Gemini API (Cloud AI)
- Maps: Leaflet.js
- Database: SQLite

---

## 🧠 AI Architecture

- User Input → Prompt Engineering
- AI Processing:
  - Local Model (LLaVA)
  - Cloud Model (Gemini)
- Output Formatting → UI Display

---

## 📊 Dashboard

- Total disasters tracked

## 🗂 History
